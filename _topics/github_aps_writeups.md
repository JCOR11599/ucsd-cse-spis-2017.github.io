<h2 style="display:block;text-align:left">Editing a Markdown File</h2>
</div>
In the github.com interface, any filename that ends with .md is treated as a "Markdown" file. &nbsp;</div>
<div style="display:block;text-align:left"><br>
</div>
<div style="display:block;text-align:left">Markdown is a way of formatting text for web pages that is designed to be than HTML (HTML is a "Markup" language—"Markdown" is a pun on "Markup".)</div>
<div style="display:block;text-align:left"><br>
</div>
<div style="display:block;text-align:left">For the most part, in a Markdown file, you just type normal text, leaving a blank line between paragraphs. &nbsp;<br>
<br>
There are a few special characters though. &nbsp;This is not a complete list—these are just a few examples.<br>
<br>
</div>
<div style="display:block;text-align:left"><br>
</div>
<div style="display:block;text-align:left">
<table border="1" bordercolor="#888" cellspacing="0" style="border-collapse:collapse;border-color:rgb(136,136,136);border-width:1px">
<tbody>
<tr>
<td>&nbsp;<b>What you want to do<span>&nbsp;&nbsp; &nbsp;<span>&nbsp;&nbsp; &nbsp;</span></span></b></td>
<td><b>&nbsp;How you type it</b><span><b>&nbsp;&nbsp; &nbsp;</b><span>&nbsp; &nbsp;</span></span></td>
</tr>
<tr>
<td>&nbsp;Headings<br>
</td>
<td><code># This is a heading</code></td>
</tr>
<tr>
<td>&nbsp;Make some <b>bold</b> text</td>
<td>&nbsp;<code>Make some **bold** text</code></td>
</tr>
<tr>
<td>&nbsp;Make some italic text</td>
<td>&nbsp;<code>Make some *italic* text</code></td>
</tr>
<tr>
<td>&nbsp;Insert the lowercase greek letter α</td>

<td>&nbsp;<code>Insert the capital greek letter &amp;Phi;</code></td>

</tr>

<h2>What can you do in a Markdown file?</h2>
</div>
<div style="display:block;text-align:left">Suppose you want a bullet list such as this one:</div>
<div style="display:block;text-align:left">
<ul><li>First, you do this.</li>
<li>Then you do that.</li>
<li>Then, finally something else</li></ul>
<div>You can make that by formatting it with stars in place of the bullets. &nbsp; (Don't leave blank spaces at the start of a line though, or it will look like code; the star followed by a space should be the first thing on the line.)</div>
<div><br>
</div>
</div>
<div class="sites-codeblock sites-codesnippet-block">
<div style="display:block;text-align:left">
<div><span style="font-size:13.3333330154419px;background-color:transparent"><code>* First, you do this.</code></span><br>
<span style="font-size:13.3333330154419px;background-color:transparent"><code>* Then you do that.</code></span><br>
<span style="font-size:13.3333330154419px;background-color:transparent"><code>* Then, finally something else</code></span></div>
</div>
<div style="display:block;text-align:left"></div>
</div>
<br>

You can learn a lot more about the things you can do in Markdown here at this guide:&nbsp;<a href="https://help.github.com/articles/markdown-basics/">https://help.github.com/articles/markdown-basics/</a></div>
